Setting Loon Config

Read More

Truy cập vào link dưới đây, sau đó hãy lấy địa chỉ "My Public IPv4 is" của bạn đưa cho chúng tôi để được kích hoạt VIP.

get ip address

Step 1: Khởi chạy Loon trên thiết bị iOS, kích hoạt 3 mục Rewrite, MiTM, Script.

Step 2: Profile -> Download URL và Paste Link mình đã gửi cho bạn vào như dưới hình. Sau đó chọn Sure và Sure để tải Config.

Step 3: Cài Chứng chỉ. Tại MITM chọn Certificate Management -> Chọn Install CA Certificate.

  • Cài đặt chứng chỉ:  Cài đặt (iPhone/iPad) -> Cài đặt chung ->  Cài đặt hồ sơ cấu hình/
  • Bật chứng chỉ: Cài đặt (iPhone/iPad) -> Cài đặt chung -> Giới thiệu -> Cài đặt tin cậy chứng chỉ -> Bật On Loon CA xxxxxxxx.

Step 4: Quay lại Loon, chọn mục Dashboard
- Mục Global Policy -> Chọn ViP Servers
- Mục Global Direct -> Chọn Global Proxy

Step 5: Tải Plugin. Chọn mục Policy dưới Tabs Loon -> Tại mục ViP Server chọn Actice ViP. Tiếp theo chọn Profile -> Plugin -> ấn vào icon như hình để tải Plugin. 

Step 6: Sau khi tải xong Plugin. Bạn quay lại mục Policy -> ViP Server và chọn về server Home để có thể sử dụng internet và các dịch vụ bình thường. Start Loon and Enjoy.

policies

  • ViP Server: Bao gồm Server Hack App Premium, Home - Server riêng của bạn và một số Server Fake IP khác...
  • Advertising: Chặn quảng cáo trên Safari + App Youtube và một số ứng dụng khác trên iOS.
  • Facebook ADS: Chặn quảng cáo trên Facebook.
  • OTA Apple: Chặn cập nhật phần mềm trên iOS
  • Tại ViP Server, để tránh phát sinh nên để mặc định là Home. Các server khác có dịch vụ và trang web đã bị hạn chế.
