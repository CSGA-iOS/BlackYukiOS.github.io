Loon

Giới thiệu : Loon là một công cụ mạng mạnh mẽ cho nhà phát triển. Loon hỗ trợ máy chủ proxy HTTP / HTTPS hiệu suất cao. Nhiều giao thức proxy được hỗ trợ để giúp bạn cải thiện bảo mật và quyền riêng tư trên internet.

Tính năng:
  • Hỗ trợ giao thức ss, ssr, vmess, trojan,
  • Capture tất cả lưu lượng HTTP / HTTPS / TCP / UDP
  • Record HTTP / HTTPS yêu cầu và tiêu đề header và body.
  • Bao gồm các quy tắc proxy tùy chỉnh.
  • Sử dụng javaScripts để mở rộng chức năng proxy.
  • Chặn quảng cáo theo quy tắc tên miền.
  • Hỗ trợ địa chỉ IPv6.
  • Đo lưu lượng sử dụng chính xác.

Quantumult X

Giới thiệu: Quantumult X là một công cụ mạng mạnh mẽ dành cho các nhà phát triển web và người dùng cần tùy chỉnh proxy của họ. Phiên bản này được xây dựng lại hoàn toàn từ đầu, và bao gồm rất nhiều tính năng mới tuyệt vời.

Tính năng:
  • Có thể ghi lại toàn bộ yêu cầu và phản hồi HTTP.
  • Giải mã HTTP MitM có thể hoạt động cho giao dịch từ giao diện TUN.
  • Viết lại HTTP với chuyển hướng URL 302/307 hoặc yêu cầu sửa đổi.
  • Cài đặt DNS tùy chỉnh cho các tên miền cụ thể (IPv4 hoặc IPv6)
  • Hỗ trợ Shadowocks với các plugin obfs-tls hoặc obfs-http.
  • Hỗ trợ Shadowocks trên websocket và tls.
  • Hỗ trợ chuyển tiếp UDP nếu máy chủ hỗ trợ.
  • Hỗ trợ các chính sách khác nhau cho yêu cầu mạng.
  • Nắm bắt tất cả lưu lượng TCP và chuyển hướng theo ý bạn
  • Là một công cụ phát triển web thuận tiện trên thiết bị di động
  • Ghi nhật ký tất cả tiêu đề yêu cầu và tiêu đề phản hồi HTTP (S)
