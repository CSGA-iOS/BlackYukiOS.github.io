Hướng dẫn cài đặt data game

  • Bước 1: Vào List Game, chọn game bạn muốn chơi và tải về. Sau đó, cắm iPhone/iPad vào PC/Laptop. Chọn Manager Apps -> Mục Device
  • Bước 2: Ở mục Device, tìm game muốn cài. Click chuột phải vào game cần chọn và chọn Restore Apps Data

Chọn Choose và chọn file game cần restore. Sau đó ấn Done -> Continue để bắt đầu cài data vào iPhone/iPad.

Chú ý: Phải tắt Find My Phone trên thiết bị iPhone/iPad để cài đặt data.

  • Bước 3: Chờ cài đặt. Sau khi xong như hình

Trên iPhone/iPad: mở khoá màn hình và để máy tự động cài đặt. Sau khi hoàn tất, mở Game lên kiểm tra data đã có hay chưa. (Một số trường hợp bị lỗi tự động cài đặt, phải cài đặt thủ công - Chú ý phần chuyển đổi dữ liệu, chọn "Không chuyển đổi dữ liệu")
